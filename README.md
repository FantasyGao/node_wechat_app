 ### express+node-wechat 微信公众号 
 
 #### 功能概述：
 1. 自定义多级菜单
 2. 客服系统（socket模块实现一对一实时对话）
 3. 高德地图调用
 4. 留言问题存库（node-mysql模块实现）
 5. 爬取健康网数据（superagent+cheerio模块实现）
 
 #### 用法：
 
  ``` shell
  git clone https://github.com/FantasyGao/node_wechat_app & cd node_wechat_app //clone文件到本地；
  
  npm install //完成安装
  
  node start 
 ```
